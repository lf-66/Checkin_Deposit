- 设定个人目标与所需金额

- 管理零花钱的存入和取出
- 为目标进行专项储蓄
- 完整的交易记录和统计

### 📊 历史记录
- 日历视图展示打卡记录
- 统计打卡完成情况
- 追踪习惯养成进度

- 数据存储在本地，保护隐私
- 支持 JSON 文件导入导出
- IndexedDB 本地持久化存储

## 🚀 快速开始

### 环境要求
- 现代浏览器（推荐 Chrome/Edge）
- 支持 File System Access API

### 本地运行

1. **克隆仓库**
   ```bash
   git clone https://github.com/yourusername/checkin-deposit.git
   cd checkin-deposit
   ```

2. **启动应用**
   
   方式一：使用本地服务器
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```
   
   方式二：直接打开
   - 在浏览器中直接打开 `index.html` 文件

3. **访问应用**
   - 打开浏览器访问 `http://localhost:8000`
   - 或直接双击打开 `index.html`

## 📖 使用指南

### 首次使用

1. 打开应用后，点击底部导航栏的 **⚙️ 设置**
2. 点击 **"加载最新数据"** 按钮
3. 选择包含您数据的文件夹
4. 应用会自动加载最新的 JSON 数据文件

### 创建打卡计划

1. 点击底部导航栏的 **🎯 目标**
2. 在 **"打卡计划管理"** 区域点击 **"添加计划"**
3. 填写计划名称、内容和每日奖励钻石数
4. 设置连续打卡奖励（可选）
5. 点击 **"添加"** 保存

### 每日打卡

1. 在 **📝 打卡页** 查看今日打卡计划
2. 点击计划右侧的按钮完成打卡
3. 完成打卡后会自动获得钻石奖励
4. 查看顶部的钻石数量变化

### 设置目标

1. 在 **🎯 目标页** 的 **"我的目标"** 区域
2. 点击输入框输入目标名称
3. 点击 **💎** 图标设置目标价格
4. 按回车或点击 **"+"** 添加目标

### 兑换钻石

1. 点击底部导航栏的 **💰 兑换**
2. 输入要兑换的钻石数量
3. 查看可获得的现金金额
4. 点击 **"兑换"** 完成操作

### 管理储蓄

1. 点击底部导航栏的 **💵 储蓄**
2. **存入**: 选择存入类型和目标，输入金额和理由
3. **取出**: 输入取出金额和理由（必须填写）
4. 查看交易记录和资产统计

### 查看历史

1. 点击底部导航栏的 **📊 历史**
2. 在日历中查看每日打卡情况
3. 绿色表示全部完成，红色表示部分完成
4. 点击日期查看详情

### 数据备份

1. 点击底部导航栏的 **⚙️ 设置**
2. 点击 **"导出数据"** 按钮
3. 选择保存位置，文件将以 `YYYY-MM-DD.json` 格式保存
4. 建议定期备份数据文件

## 🎮 奖励机制

### 每日打卡奖励
每个计划完成打卡可获得设定的钻石奖励。

### 连续打卡奖励
| 连续天数 | 额外奖励 |
|---------|---------|
| 3天     | 10钻石  |
| 7天     | 20钻石  |
| 15天    | 50钻石  |
| 30天    | 100钻石 |

### 默认打卡计划
| 计划名称 | 内容 | 每日钻石 |
|---------|------|---------|
| 每日运动 | 蹲起、仰卧起坐、平板撑等 | 10 |
| 整理日用品 | 书桌、书包整理 | 5 |
| 平板使用记录 | 周一到周四不玩平板 | 10 |
| 电视使用记录 | 周一到周四最多30分钟 | 10 |
| 零花钱使用记录 | 不乱买没用的东西 | 10 |
| 按时洗漱 | 晚上8:30前洗漱完毕 | 5 |
| 英语学习 | 每日英语学习15分钟 | 10 |
